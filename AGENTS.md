# Guia de contribuição

Este repositório contém uma skill em português para comparação de preços de componentes de PC em lojas brasileiras.

## Estrutura

- SKILL.md: instruções principais da skill.
- README.md: visão geral e uso.
- scripts/pc_price_finder.py: ponto de entrada principal em Python.
- tests/: testes de regressão.

## Regras

- Mantenha a skill em português.
- Preserve o fluxo de execução local e o formato JSON esperado.
- A documentação deve continuar simples e útil para agentes e humanos.
- Antes de enviar mudanças, rode `python -m pytest -q`.
